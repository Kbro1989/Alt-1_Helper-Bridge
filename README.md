# Alt-1_Helper-Bridge — Alt1 Game Overlay Bridge

> HTML · TypeScript · Vite · Alt1 Toolkit

Bridge between the Alt1 Toolkit overlay system and the POG2 backend.
Vite-built HTML app that runs as an Alt1 app, forwarding game events to
Cloudflare Workers for processing.
